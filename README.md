# Phishing Email Detection & Awareness System

## Objective

The purpose of this project is to analyze phishing email samples, identify phishing indicators, classify risks, and provide cybersecurity awareness guidelines for users and organizations.

## Tools Used

* Google Admin Toolbox Message Header Analyzer
* MXToolbox Email Header Analyzer
* VirusTotal
* Microsoft Word
* GitHub

## Methodology

1. Collected phishing email samples.
2. Analyzed sender details, links, attachments, and content.
3. Identified phishing indicators.
4. Classified risk levels.
5. Created awareness and prevention guidelines.

## Email Samples Analyzed

* Account Verification Scam
* Invoice Payment Scam
* Password Reset Scam

## Risk Classification

All analyzed emails were classified as Phishing (High Risk) due to multiple phishing indicators such as suspicious links, urgency tactics, and generic greetings.

## Prevention Guidelines

* Verify sender addresses.
* Avoid clicking unknown links.
* Do not open unexpected attachments.
* Enable Multi-Factor Authentication (MFA).
* Report suspicious emails immediately.

## Repository Structure

samples/ - Email samples used for analysis

screenshots/ - Evidence and report screenshots

README.md - Project overview

## Conclusion

Phishing remains one of the most common cyber threats. User awareness and proper verification practices significantly reduce the risk of successful phishing attacks.
